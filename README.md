
  # Scrapping C1 KPU Pemilu Presiden 2024 📝  
  Kode ini dapat digunakan untuk scrapping data C1 dari website KPU dapat digunakan untuk mengawal proses Pemilu 2024.  
  
  ## Requirement 🚀  
  - Node Js version 18
  
  
## How To Use  🖥️
Install Puppeteer

~~~bash  
  npm install puppeteer
~~~

Run code 

~~~bash  
node scrappingc1.js
~~~
